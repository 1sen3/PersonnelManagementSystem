## 项目简介
本项目是一个基于 WinUI3 + MySql 的人事管理系统，支持人事部门管理人员进行人事管理工作，包括招聘、解聘、辞职、换岗、晋升等功能。
## 技术栈
- **开发框架**：WinUI 3
- **开发语言**：C#、XAML
- **数据库**：MySQL
- **开发工具**：Visual Studio 2022、Visual Studio Code、MySQL Workbench
